# AI-Generated Text Detection System

A Generative AI course project — a full-stack machine learning system that classifies whether a piece of text was written by a human or generated by an AI language model. It explores both classical feature-engineering approaches and fine-tuned transformer models.

---

## Overview

As large language models become ubiquitous, detecting AI-generated content is an increasingly important problem. This project tackles it from two angles:

| Track | Approach | Models |
|-------|----------|--------|
| Classical ML | Statistical + linguistic features | Logistic Regression, SVM, XGBoost |
| Deep Learning | Fine-tuned transformers | RoBERTa-base, DistilBERT-base |

The deployed system uses a fine-tuned **RoBERTa** model hosted on Hugging Face Hub, served through a **FastAPI** backend and a **Streamlit** web UI.

---

## Features

- Binary classification: **Human** vs **AI-Generated** text
- Confidence score and per-class probability breakdown
- REST API endpoint (`/predict`) for programmatic access
- Interactive web UI for manual testing
- Dockerized deployment (Hugging Face Spaces compatible)
- Adversarial robustness testing pipeline
- Classical ML baseline with interpretable features (perplexity, readability, naturalness)

## Getting Started

### Prerequisites

- Python 3.9+
- pip

### Installation

```bash
git clone https://github.com/HamzaSidd17/GenAI-Project.git
cd GenAI-Project/Deployment
pip install -r requirements.txt
```

### Run the API

```bash
uvicorn app:app --host 0.0.0.0 --port 7860
```

The API will be available at `http://localhost:7860`.

### Run the Streamlit UI

```bash
streamlit run streamlit_app.py
```

### Docker

```bash
docker build -t textsentinel .
docker run -p 7860:7860 textsentinel
```

## API Reference

### `POST /predict`

Classify a piece of text.

**Request body:**
```json
{
  "text": "Your text to classify here."
}
```

**Response:**
```json
{
  "prediction": "AI",
  "confidence_score": 0.9732,
  "probabilities": {
    "human": 0.0268,
    "ai": 0.9732
  }
}
```

### `GET /`

Health check — returns `{"status": "running", "model": "RoBERTa-FineTuned"}`.

## Model

The deployed model is a fine-tuned `roberta-base` for binary sequence classification, hosted publicly on Hugging Face Hub:

> **[Hamza-003/ai-text-detector-roberta](https://huggingface.co/Hamza-003/ai-text-detector-roberta)**

Training involved:
- Dataset preprocessing and balancing (human vs AI-generated text pairs)
- Feature engineering for the classical ML baseline
- Fine-tuning on the sequence classification task
- Adversarial testing and robustness evaluation
